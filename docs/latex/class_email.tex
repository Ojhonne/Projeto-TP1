\doxysubsection{Detailed Description}
Representa o formato valido para um endereco de email. 

Um email eh composto por duas partes separadas pelo caractere \textquotesingle{}@\textquotesingle{}(parte-\/local@dominio) e atua como identificador de uma entidade.

Parte local:\+
\begin{DoxyItemize}
\item Pode conter letra (a-\/z), digito (0-\/9), ponto (.) ou hifen (-\/).
\item Nao pode iniciar ou terminar com ponto ou hifen.
\item Ponto ou hifen deve ser seguido por letra(s) ou digito(s).
\item Comprimento maximo de 64 caracteres.
\end{DoxyItemize}

\doxylink{class_dominio}{Dominio}:\+
\begin{DoxyItemize}
\item Composto por uma ou mais partes separadas por ponto (.).
\item Cada parte pode conter letra (a-\/z), digito (0-\/9) ou hifen (-\/).
\item Nao pode iniciar ou terminar com hifen.
\item Comprimento maximo de 255 caracteres.
\end{DoxyItemize}

Regras adicionais:\+
\begin{DoxyItemize}
\item Deve existir exatamente um caractere \textquotesingle{}@\textquotesingle{} separando as partes.
\item Parte local e dominio nao podem ser vazios. 
\end{DoxyItemize}
