\doxysubsection{Detailed Description}
Representa uma data válida no formato D/\+\+MM/\+\+AAAA ou DD/\+\+MM/\+\+AAAA. 

A classe \doxylink{class_data}{Data} herda as operações da classe abstrata \doxylink{class_dominio}{Dominio} e gerencia valores cronológicos compreendidos entre os anos de 2000 e 2099. Ela encapsula regras de validação para dias, meses, anos (incluindo anos bissextos) e impede formatos inconsistentes (como dias iniciados em zero \textquotesingle{}01/\+\textquotesingle{} e meses que nao estao no formato \textquotesingle{}MM\textquotesingle{}).
