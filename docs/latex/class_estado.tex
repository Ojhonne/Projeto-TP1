\doxysubsection{Detailed Description}
Representa um estado válido de andamento de tarefa. 

A classe \doxylink{class_estado}{Estado} herda as operações públicas da classe abstrata \doxylink{class_dominio}{Dominio}. A classe \doxylink{class_estado}{Estado} é responsável por garantir que o usuário defina um estado válido, que por sua vez, são eles:\+ "{}\+A FAZER"{}, "{}\+FAZENDO"{}, "{}\+FEITO"{}.

OBSERVAÇÃO:\+ O estado deve ser EXATAMENTE igual a uma das 3 alternativas, sem letras minúsculas!
