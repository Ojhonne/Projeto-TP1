\doxysubsection{Detailed Description}
Representa o formato válido para armazenar um \doxylink{class_texto}{Texto}. 

A classe domínio \doxylink{class_texto}{Texto} herda as operações públicas da classe abstrata \doxylink{class_dominio}{Dominio}. ~\newline
A classe \doxylink{class_texto}{Texto} representa palavras que formam uma maneira de transmitir uma mensagem com regras formais de espaçamento e caracteres permitidos.

Regras:\+
\begin{DoxyItemize}
\item Tamanho de até 40 caracteres.
\item Não pode começar ou terminar com espaço em branco, virgula ou ponto.
\item Espaço em branco é deve ser seguido por letra ou digito.
\item Ponto não pode ser seguido por virgula ou ponto.
\item Virgula não pode ser seguido por virgula ou ponto.
\item Apenas letras, espaços, digitos, ponto e virgula são permitidos. 
\end{DoxyItemize}
