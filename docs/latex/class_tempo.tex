\doxysubsection{Detailed Description}
Representa o formato válido para armazenar um \doxylink{class_tempo}{Tempo}. 

A classe domínio \doxylink{class_tempo}{Tempo} herda as operações públicas da classe abstrata \doxylink{class_dominio}{Dominio}. ~\newline
A classe \doxylink{class_tempo}{Tempo} representa um valor numérico que determina a duração dos eventos com regras formais de limties e caracteres permitidos.

Regras:\+
\begin{DoxyItemize}
\item O tamanho minimo é 1.
\item O tamanho maximo é 365.
\item Apenas números são permitidos. 
\end{DoxyItemize}
