\doxysubsection{Detailed Description}
Representa o formato válido para armazenar uma prioridade do usuário. 

A classe domínio \doxylink{class_prioridade}{Prioridade} herda as operações públicas da classe abstrata \doxylink{class_dominio}{Dominio}. ~\newline
A classe \doxylink{class_prioridade}{Prioridade} e responsável por garantir que cada usuário receba uma prioridade ao acessar o programa.
