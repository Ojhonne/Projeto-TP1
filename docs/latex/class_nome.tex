\doxysubsection{Detailed Description}
Representa o formato válido para armazenar um \doxylink{class_nome}{Nome}. 

A classe domínio \doxylink{class_nome}{Nome} herda as operações públicas da classe abstrata \doxylink{class_dominio}{Dominio}. ~\newline
A classe \doxylink{class_nome}{Nome} representa palavras que distinguem e identificam uma coisa própria com regras formais de espaçamento e caracteres permitidos.

Regras:\+
\begin{DoxyItemize}
\item Tamanho de até 10 caracteres.
\item Não pode ser vazio.
\item Não pode começar ou terminar com espaço.
\item Espaço em branco é deve ser seguido por letra.
\item Cada termo deve iniciar com letra maiúscula ou minúscula.
\item Apenas letras e espaços são permitidos. 
\end{DoxyItemize}
