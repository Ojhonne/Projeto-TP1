\doxysubsection{Detailed Description}
Representa um codigo identificador valido. 

A classe \doxylink{class_codigo}{Codigo} herda da classe abstrata \doxylink{class_dominio}{Dominio} e representa um identificador de entidade no formato LLDDD, em que:\+


\begin{DoxyItemize}
\item L representa uma letra maiuscula de A a Z.
\item D representa um digito de 0 a 9.
\end{DoxyItemize}

Regras de validacao:\+
\begin{DoxyItemize}
\item O codigo deve possuir exatamente 5 caracteres.
\item Os dois primeiros caracteres devem ser letras maiusculas.
\item Os tres ultimos caracteres devem ser digitos. 
\end{DoxyItemize}
