\doxysubsection{Detailed Description}
Representa o formato válido para armazenar uma senha. 

A senha deve ter exatamente 6 caracteres, eles podem ser:\+
\begin{DoxyItemize}
\item Letras de A -\/ Z podendo ser maiúsculas ou minúsculas;
\item Digitos de 0 -\/ 9;
\end{DoxyItemize}

Para uma senha ser considerada válida ele deve seguir as seguintes obrigatoriedades:\+
\begin{DoxyItemize}
\item Letra não pode ser seguida por letra;
\item Dígito não pode ser seguido por dígito;
\item Existe pelo menos uma letra minúscula (a-\/z);
\item Uma letra maiúscula (A-\/Z);
\item Um dígito (0-\/9); 
\end{DoxyItemize}
