Descrição detalhada

Classe base abstrata que padroniza o comportamento de classes de domínio.

A classe Dominio gerencia o armazenamento de uma string e garante que apenas valores válidos sejam atribuídos, delegando a regra de validação para as classes derivadas (filhas) através de um método virtual puro.
