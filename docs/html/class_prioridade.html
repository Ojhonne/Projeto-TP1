Detailed Description

Representa o formato válido para armazenar uma prioridade do usuário.

A classe domínio Prioridade herda as operações públicas da classe abstrata Dominio.
A classe Prioridade e responsável por garantir que cada usuário receba uma prioridade ao acessar o programa.
